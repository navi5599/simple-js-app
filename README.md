# Simple POKEDEX JS App
***
## Introduction
---
This is a simple Pokedex app.It shows an list of 150 Pokemons loaded from external public API : [Pokemon API](https://pokeapi.co/api/v2/pokemon/?limit=150).

---
## Technologies
---
* `HTML`
* `CSS`
* `Javascript`
* `Bootstrap v4.3.1`
